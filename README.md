### Installation

1. Get your API key from your Google Cloud Platforms (it's a single string, and make sure you have enabled Maps API) [https://console.cloud.google.com/apis/credentials](https://console.cloud.google.com/apis/credentials)
2. Clone the repo
   ```sh
    git clone https://github.com/KevinAS28/SETH.git
    cd SETH
   ```
3. Install the requirements (it's also best practice to create a Venv first)
   ```sh
    python -m pip install -r requirements2.txt
   ```
4. Enter your API key in `SETH/kevin_api_key` (without new line at the end)
   ```sh
   ~/SETH$ cat SETH/kevin_api_key 
AIzaSyCokW0nG....
   ```
5. Make sure you have allowed firewall port rules (we are using 8000, 8080)
